## Overview
### About

[EliteEdge](https://EliteEdge-frontend.vercel.app) is a MERN fullstack furniture Ecommerce project that has the following features:

- Registeration / logging-in system
- Shopping cart
- Product details
- Checkout process
- User account
- Admin dashboard
- Search for products
- Filter products by price, categoty and brand
- Contact form
